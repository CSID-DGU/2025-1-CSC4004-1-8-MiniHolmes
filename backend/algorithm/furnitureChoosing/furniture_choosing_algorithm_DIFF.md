# furniture_choosing_algorithm.py 변경점 요약 (원본 대비)

## 1. 데이터 구조/클래스
- `_id` 필드로 MongoDB 연동, 일부 필드 optional 처리 (`get()` 사용)
- `to_dict()` 메서드 추가 (JSON 변환/응답용)

## 2. 등급/로직
- **크기 등급/가격 등급**: 카테고리별 상세 기준 주석 추가, 등급 분류 로직 수정
- **추천 세트**: 3회 반복, 선택된 가구는 available_items에서 제외 유지
- **가구 선택**: 점수 계산(스타일, 컬러톤, 크기, 가격) 방식 수정
- **정렬/선택**: 점수순 정렬 후 최상위 선택, 크기/예산 초과 시 교체 로직 유지

## 3. 예외처리/로깅
- 가격 정보 없는 경우 기본값 반환 등 예외처리 추가
- 상세 로깅: 점수 계산, 선택/교체 이유, 경고 등 로그 기록
- 선택 이유/교체 이유를 selection_reasons, logger로 남김

## 4. 입출력/실행 구조
- `main()` 함수 추가: 표준 입력/출력, JSON 입출력, 예외처리, 전체 파이프라인 구현
- 결과를 JSON으로 출력, 오류 발생 시 로깅 및 종료
